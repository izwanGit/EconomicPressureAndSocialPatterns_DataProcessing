Economic Pressure & Social Patterns
This project examines how economic disparities influenced social stability and crime patterns in Malaysia. It utilizes Python to clean and analyze household income (HIES) and crime statistics.

Repository Structure
Data Cleaning:
- crime_data_cleaning.py - Preprocessing script for district crime rates.
- hies_data_cleaning.py - Preprocessing script for Household Income & Expenditure Survey data.

Datasets:
- crime_district.csv - Raw crime statistics by district.
- hies_state.csv - Household income data by state.

Tech Stack
- Python
- Pandas / NumPy
- Data Preprocessing & Analysis
